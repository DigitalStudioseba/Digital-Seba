✏️ NID Correction Guide

Name Correction Documents

  • ✅ Birth certificate (with correct name)
  • ✅ SSC/HSC certificate
  • ✅ Affidavit from Notary Public
  • ✅ Updated NID application form

Date of Birth Correction

  • ✅ Birth certificate (original)
  • ✅ School certificate showing DOB
  • ✅ Court order (if required)

Address Correction

  • ✅ Utility bill (electricity/gas/water)
  • ✅ Union Parishad chairman certificate
  • ✅ House ownership documents
